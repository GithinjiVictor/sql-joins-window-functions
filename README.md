# SQL Sales & Inventory Assignment

## Overview
This project contains SQL scripts for a Sales and Inventory management system.

The assignment covers:

- Database schema creation
- Data insertion
- Aggregate functions
- Joins (INNER, LEFT)
- Grouping and filtering
- Window functions
- Advanced SQL queries

## Database Used
PostgreSQL

## File Included
- sales_inventory_assignment.sql  
  Contains:
  - CREATE TABLE statements
  - INSERT statements
  - 50 SQL queries answering the assignment questions

## Concepts Practiced
- Primary and Foreign Keys
- GROUP BY and HAVING
- Aggregate functions (SUM, COUNT, AVG, MAX, MIN)
- Subqueries
- Self Joins
- Window Functions (ROW_NUMBER, RANK, etc.)
